5

Periodical

Info on cover/title page/verso indicates work is a serial publication or periodical, possibly an extracted volume cataloged as a monograph.

 

Terms to watch out for: Journal; Magazine; Bulletin. Look for some indication of periodicity (e.g., seasons, months, years).

 

The presence of a number on a work, by itself, does not qualify the work as a periodical (e.g., numbered monographic series are not likely to be registered/qualify as periodicals for purposes of copyright review.

Periodicals can include separately copyrighted works. Also works registered and renewed as periodicals (or contributions to periodicals) would not be not included in the Stanford Renewal Database.

Code: und/nfi

Note Category: Periodical

Note title, vol., issue, date (if known), and any other pertinent details

6

Foreign Work

Evidence that work in hand is a foreign publication (primary place of publication on piece is not US); or  was first/previously published outside US.

 

Note: If title page/verso says (literally) "published simultaneously" in another country, consider it a US work.

Note: if work includes statement referencing international copyright (e.g. Berne Convention or International Copyright Union), look for evidence that work was previously published outside the U.S., otherwise ignore and consider it a U.S. work.

Foreign works are not necessarily subject to US copyright formalities and require additional information to properly assess status.

8

Publication Date pre-1923

Title page/verso or other frontmatter indicates work was published prior to 1923 (in the US).

 

Note: if there is only a copyright date listed, and no date of publication, consider the copyright date as date of publication.

Works published prior to 1923 in the US are in the public domain.

Code: pd/cdpp

Note Category: Date

Note actual publication date as printed on the piece. (If there is only a copyright date, cite the copyright date.  Use "c" in front of copyright date. See example.)

•pub 1912 / 1919 printing;

•c1917 (only date on pub);

   

9

Publication Date post-1963

Title page/verso or other frontmatter indicates work was published 1964 or later.

 

This situation may also  arise when a volume is part of a multivolume set (all on one bib record), as the date displayed will be for the first volume in the set, so the volume being reviewed may have a later publication date outside the range of 1923-1963.  

 

Note: if there is only a copyright date listed, and no date of publication, consider the copyright date as date of publication.

Works published after 1963 in the US are not subject to the renewal requirement, so should be considered in copyright.

Code: ic/cdpp

Note Category: Date

Note actual publication date as printed on the piece. (If there is only a copyright date, cite the copyright date.) If the work is part of a multivolume set, note the volume number. Use "c" in front of copyright date. See example.

•1966 pub;

 

•1990; c1973 (v.14)

10

Not Class A*

Work is a standalone work or includes such works that would not be in the Stanford Renewal Database.

Examine title page, verso, preface, and introduction for indication that work is or includes any of the following:

serials/periodicals; contributions to periodicals (articles), addresses for oral delivery (i.e. lectures, sermons, speeches); dramatic works (i.e. plays); musical works; music scores; song lyrics, maps and atlases; and works of art, which includes art reproductions, scientific and technical drawings, photographic works, prints, and pictorial illustrations.

*See Not Class A Help under "Search Help Documentation."

We cannot determine whether the copyright was renewed if it is a work that would not be included in the Stanford database.

We can base our determination on copyright status of original work and ignore the date when the reprint/photocopy was produced.

Work based on previous work: new material added (Edition)

Title page/verso indicate work is a new edition of a previously published work (not necessarily with the same title), or there is indication that the work is based on an earlier work and new material (e.g., introduction, preface, forward) has been added or the contents has been updated or revised.

We must base our final determination on the renewal status of both the original work and the new work

1. Check for renewal of work in hand, if renewal is found, enter renewal id and date, code ic/ren.

2. If renewal not found for work in hand, check for renewal of any works published previously (but no earlier than 1923) upon which the work in hand is based.

a) If no renewal found, code pd/ren.

 b) If renewal is found for previous work,  enter renewal ID, get date, code ic/ren; Edition.

 

Note: Renewals in works published subsequent to work in hand are irrelevant to status of the earlier work so can be ignored.)

Note: Also see note in Reprint instructions above.

15

Renewal found in Stanford DB; check for copyright notice as last resort

Renewal located for work in hand but there is no copyright notice visible anywhere on the piece.

 

The copyright notice must have the symbol © or the word Copyright (or its abbreviation Copr.) followed by the date, and the name of the copyright holder.

If the work was renewed but there is no copyright notice visible on the digitized volume, the work is in the public domain.

Check for presence of copyright notice, which is most often displayed on the tp verso, but if not found, check carefully all front matter including first page of text.

 

Note: if it seems that pages are missing from frontmatter, follow instructions for Missing in step 1.

1. If notice found, enter renewal ID and date (see step 14 above), code ic/ren.

2. If no copyright notice found, code pd/ncn
