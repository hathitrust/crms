CRMS Reviewer Help

(Last Revised 11-12-2009)

Interface Notes

  • Currently, the CRMS is only known to be fully compatible with Firefox 3.x browser.
  • Safari and Internet Explorer can be used, but for security reasons will generate an error on the HathiTrust frame unless:
    • The user has previously logged in to HathiTrust.
    • (IE only) the user lowers IE's security level. (The security level can immediately be restored after a page refresh, just to get the HT cookie.)
  • Internet Explorer can open help, use cases, etc. popup windows as windows rather than tabs. To fix:
    1. Go to Tools → Internet Options → General.
    2. Click on the "Settings" button under "Tabs".
    3. Find the "When a pop-up is encountered:" section and click "Always open pop-ups in a new tab".
  • Safari for Mac does this too. To fix (on Safari 3.1 and later at least):
    1. Run the Terminal application at /Applications/Utilities/Terminal.
    2. Paste in the command "defaults write com.apple.Safari TargetedClicksCreateTabs -bool true" and hit return.
    3. Restart Safari.
  • Google Chrome seems compatible and will be undergoing continuing tests.
  • The zoom feature within Firefox can create misalignment of the review interface. If your interface looks strangly aligned, go to the Firefox menu and click View → Zoom → Reset to restore the page's default viewing size. By default, Firefox saves a history of whatever zoom setting you used last on every cached web page. This zoom history feature can be turned off within the Firefox core config but see the Mozilla site for details.
  • Double-clicking and dragging the Renewal ID vs. copy and pasting it. We've determined that the "drag method" is the most efficient way (a way which is less likely to gather in any odd characters) to get the Renewal ID from the Stanford web site to the CRMS. To execute the drag method, double-click on the Renewal ID (this highlights the Renewal ID) and then hold the left mouse button down while hovering over the highlighted Renewal ID and drag it over to the CRMS Renewal ID input box and drop it in by letting go of the left mouse button.
