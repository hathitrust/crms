Note: Stanford University's Copyright Renewal Database covers renewals for Class A materials only. These materials have a registration number with the letter A as a prefix. The database does not include renewals for all other Classes (B-M) of materials. See chart below for more information on Stanford's coverage.

Special Use Case
Instructions
Comments on Note Field

1s
Title page/verso says (literally) “published simultaneously” in another country

Consider US work and check for Copyright notice, renewal (Use Cases 1/2/3 above)
Note MISC and the country plus that the pub is simultaneous
MISC: British pub simultaneous
2s
Title page/verso indicates work is a translation of a foreign publication
Treat as foreign pub (Use Cases 9/10/11/12 above) but note as translation
Note TRANSLATION and place of original pub (foreign title if known)
TRANSLATION: Spain “San Cristobal de la Habana”
3s
Title page/verso indicates work is a Dissertation or Thesis
Check for renewal. If renewed, code ic/ren. If no renewal, code und/nfi and make a note.
Note DISSERTATION/THESIS; if not renewed, note "no renewal," and also note whether there is or is not a copyright notice.

DISSERTATION/THESIS: no renewal; no copyright notice

DISSERTATION/THESIS: no renewal; includes copyright notice
4s
Title page/verso indicates work is part of a numbered Monographic Series

Check books for copyright notice and renewal of monograph title. Series receives no special treatment.

When reviewing numbered series, be watchful for any indication that the work might be copyrighteed as a Serial/Periodical (see case no. 6s) or is a Collection of works (i.e., articles) that could be separately copyrighted (see case no. 7s).

N/A

5s
Title page/verso indicates work is part of an unnumbered Monographic Series
Check for copyright notice and renewal of monograph title. Series receives no special treatment.
N/A
N/A
6s
Title page/verso indicates work is a Serial/Periodical, (i.e.,an extracted volume cataloged as a monograph)

This material will use terms like Bulletin, Journal, or Magazine to indicate that it's a Serial/Periodical that was cataloged as a monograph. Code these und/nfi and make a note.

These are what are known as "Non-Class A" materials (see note at top of Use Cases).

Note all serial info (when known): title of serial; vol. and issue; and date.
SERIES/SERIAL: Assoc. of Casualty and Surety Co.; ED 14, no. 3, 1953
7s
Title page/verso indicates a Collection of material that includes (or could include) separately copyrighted material

Check for renewal of primary piece/collection title. If renewed, code ic/ren. If not renewed, code und/nfi (except architectural works which weren’t covered under copyright laws until 1990).

These are what are known as "Non-Class A" materials (see note at top of Use Cases).

See case no. 6s for a complete Serial/Periodical

Note COLLECTION and type of works and any info on tp/verso about copyright of individual works.

Examples of collections: Artwork, Lectures, Proceedings, Poetry, Plays, and Maps or Atlases.

 

COLLECTION: articles from Foreign Affairs

COLLECTION: reproductions of copyrighted artwork

 
8s Title page/verso indicates work is a standalone lecture, sermon, speech, play, or poem

Check for renewal. If renewed, code ic/ren. If not renewed, code und/nfi and make a note.

These are what are known as “Non-Class A” materials (see note at top of Use Cases).

See case no. 7s for a Collection of works

Note MISC for any standalone piece of Non-Class A material.

Examples: Lecture, Play, Poem, Sermon, or Speech.
MISC: Play

13s
UN Documents
Most not in copyright but some are, and status can be difficult to determine. Confirm published in US and if yes, check for copyright notice and renewal. If you find BOTH, make it und/nfi and note what was found.
Note MISC and that it is or appears to be a “UN Doc.” Report presence of copyright notice and anything you find in Stanford database. Anne will look at these case by case.   
MISC: UN Doc (copyright notice/renewal found)  
14s
US Government Documents - publisher is the GPO (Government Printing Office) or a US Federal Government Agency

Check for copyright notice only. If there is no copyright notice, code it pd/ncn.

Important Note: ignore series, serial, proceedings, etc.

If it looks like a US Gov Doc but it has a copyright notice, code it und/nfi.

If und/nfi (because it includes a copyright notice, make a MISC note questioning its copyright status.

MISC: US Gov Doc with copyright notice?

15s Title page/verso indicates a portion of the work was reprinted from another publication.

Check for renewal of primary piece. If renewed, code ic/ren. If not renewed, code und/nfi. In either case, make a note.

See case no. 16s for whole reprints.

Note REPRINT and cite title(s) of reprinted material included. REPRINT: The Cowboy Prayer

Use the original publication to make a determination.

See case no. 15s for partial reprints.
